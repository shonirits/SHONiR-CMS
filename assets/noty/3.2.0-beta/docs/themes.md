**Available Themes**: <br> `mint`, `sunset`, `relax`, `nest`, `metroui`, `semanticui`, `light`, `bootstrap-v3`, `bootstrap-v4`

**Default Theme**: `mint`

Usage;

```javascript
new Noty({
    theme: 'themeName',
    text: 'Some notification text'
}).show();
```
